# Minha Primeira API 🚀

Este é o projeto da minha primeira API, criado com o objetivo de servir como base para meus estudos na área de desenvolvimento Backend.

## 💻 Tecnologias Utilizadas

- **C#**
- **.NET 8** (ASP.NET Core)
- **Swagger** (Swashbuckle) para documentação e teste da API

## 🎯 Objetivos de Aprendizado

- Entender os conceitos básicos de uma API RESTful.
- Aprender sobre rotas, controllers e endpoints no ASP.NET Core.
- Praticar a estruturação de projetos backend e injeção de dependência.

## 🚀 Como Executar o Projeto

1. Certifique-se de ter o [.NET 8 SDK](https://dotnet.microsoft.com/pt-br/download/dotnet/8.0) instalado na sua máquina.
2. Clone este repositório ou abra os arquivos no Visual Studio.
3. Se estiver usando o **Visual Studio**, basta clicar no botão de "Executar" (ou pressionar `F5`).
4. Se estiver usando o terminal, navegue até a pasta do projeto e execute:
   ```bash
   dotnet run
   ```
5. A aplicação iniciará. Acesse seu navegador na URL gerada (ficará visível no console e abrirá automaticamente no VS) com `/swagger` no final para visualizar a interface, onde você poderá testar os endpoints da API.
